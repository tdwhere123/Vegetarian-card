1. 项目概况 (Project Overview)
- 项目名称： 蔬蔬灵感集 (Veggie Spark)
- Slogan： “咔嚓！把蔬菜变成魔法卡片✨”
- 核心价值： 无论你是否有食材，AI都能为你提供素食灵感，并通过“集卡游戏化”机制，激励用户坚持素食烹饪。
- 目标用户： 想要尝试素食但缺乏灵感的年轻人、佛教徒（无五辛需求）、烹饪爱好者、不知道吃什么的人。
- 平台： 微信小程序 (基于 Uni-app + Vue3)。
- 视觉风格： 暖米色治愈风（默认） / 紫色魔法风（狂野模式）。

2. 核心流程 (User Flow)
1.   输入/随机： 用户输入食材（消灭库存模式） 或 直接点击随机生成（灵感模式）。
2.   生成： AI生成菜谱 + AI绘制彩铅插画（Soft colored pencil style）。
3.   挑战： 用户获得一张**“未激活（黑白）”**的卡片，查看菜谱进行烹饪。
4.   还原： 用户拍照上传成品图（调用系统相册/相机）。
5.   解锁： 卡片**“激活（变彩）”**，存入卡册，记录制作次数与时间。

3. 功能模块详解 (Functional Requirements)
 3.1 首页 (Home Page) - V2.0 更新
-    视觉风格： 默认暖米色背景，带有动态魔法粒子（蔬菜/星星）。
-    核心元素：
  - Slogan展示： 顶部可爱字体展示“Veggie Spark”。
  - 输入区域：
    - 多行文本框，placeholder提示：“今天冰箱里有什么？（例如：番茄、土豆）”。
    - 标签栏：[🚫 无五辛] [🥛 纯素] [🔮 狂野模式]。
    - **交互更新**：开启“狂野模式”后，背景变为深紫色，粒子变为魔法符号，按钮增加呼吸灯特效。
  - 生成按钮： 紧贴输入框下方，文案根据输入状态动态变化（“用这些变魔法” / “随机给我一个惊喜”）。
  - 底部导航： 自定义 TabBar，包含“灵感厨房”和“我的卡册”，切换时带有平滑动效，支持随狂野模式变色。

 3.2 卡片详情页 (Card Detail)
- 状态 A：未解锁 (New Generated)
  - 加载动画： 魔法阵旋转 + 动态文字轮播提示（“正在研磨魔法香料...”、“正在绘制美味符文...”）。
  - 卡片顶部： 黑白滤镜的 AI 美食插画。
  - 底部操作： “📸 上传成品解锁卡片”，点击调用 `uni.chooseImage` 模拟上传。
- 状态 B：已解锁 (Unlocked)
  - 交互特效： 解锁成功后自动跳转回卡册。
  - 数据记录： 记录解锁时间、制作次数。

 3.3 我的卡册 (My Collection)
- 布局： **双列瀑布流布局**，卡片间距均匀。
- 卡片展示：
  - 图片： 彩色 AI 插画。
  - 稀有度： 右上角角标 (N/R/SR)。
  - 信息栏： 菜名 + **制作次数 (如: 制作 3 次)** + **解锁日期 (如: 2023.12.08)**。
  - 视觉优化： 卡片底部增加半透明渐变遮罩，确保文字清晰可见。

4. 技术架构与数据 (Technical Specs)
 4.1 数据结构 (Data Model)
- `cardList` 存储于本地 `uni.setStorage`。
- 新增字段： `user_photos` (数组，存储本地图片路径)。
- 图片源： 默认数据已迁移至稳定的 Unsplash 源，AI 生成图片使用 API 返回 URL。

 4.2 API 接入策略 (API Integration)
- LLM (文本生成)：
  - 模型： gpt-5-mini。
  - 提示词优化： 区分“狂野模式”（魔法风）与“现实模式”（中国家常菜、N级炒菜）。
- Image Gen (生图)：
  - 模型： jimeng-4.5。
  - 风格控制： 强制指定 "Soft and artistic colored pencil drawing" 风格。
